# Festivals-Landing-Page
Learn to Muslim, Hindu, Christam, Sikh

I have developed a comprehensive web project as part of my college assignment, showcasing my skills in JavaScript, HTML5, Bootstrap, CSS, and jQuery. This project incorporates animations using AOS library and leverages Java for enhanced functionality.

Key Features:

Multireligious Pages:

The project includes dedicated pages for four major religions: Muslim, Hindu, Sikh, and Christian. Each religion has its own set of content, creating an inclusive platform.

Blog Section:

Individual pages for each religion contain detailed blog posts, fostering a deeper understanding of cultural and religious events. A total of 12 events per religion are covered.

Chatbot Integration:

A unique chatbot engages users in conversation, providing information related to ongoing events. Powered by Java, the chatbot responds dynamically to user queries.

Contact Us Page:

The Contact Us page features a user-friendly form. Upon submission, messages are sent to my email, ensuring effective communication and feedback.

Technologies Used:

Frontend:

HTML5, Bootstrap, CSS, jQuery for a responsive and visually appealing user interface.

Backend:

JavaScript and Java for enhanced interactivity and dynamic content.

How to Interact:

Explore Religions:

Visit the individual pages dedicated to each religion to access blogs and information about the respective events.

Chatbot Interaction:

Engage with the chatbot on the main page for real-time responses to your inquiries about ongoing events.

Contact Us:

Utilize the Contact Us page to share your thoughts, feedback, or inquiries. I will promptly receive your messages via email.
